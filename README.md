# Glow Plugins

Claude Code plugins for [Glow](https://talktoglow.com).

## Install

```shell
/plugin marketplace add yourorg/glow-plugins
/plugin install glow@glow-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `glow` | Connect Claude Code to Glow via MCP |

## Skills

- `/talktoglow` — Talk to Glow
